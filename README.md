## 📖 关于 Lynote (About)**Lynote** 不仅仅是一个工具，它是你的第二大脑。我们致力于解决碎片化学习的痛点，通过 AI 赋能 "输入-内化-输出-应用" 全链路。无论你是通过 YouTube 学习新技能的学生，还是需要快速处理文档的职场人，Lynote 都能帮你把信息转化为真正的认知。

### ✨ 核心功能 (Core Features)

根据我们的开发路线，以下功能已上线：

#### 🛡️ 内容理解与检测 (AI Content Integrity)
支持检测主流大语言模型生成的文本：
* **AI Detect (P0):** 高精度检测文本是否由 AI 生成（支持检测 GPT-4, Claude 3, Gemini, Llama 3 等主流模型痕迹）。



## ⚡️ 快速上手 (Quick Start)> **体验丝滑的 "打开-粘贴-完成" 工作流**
1.  **打开** [Lynote.ai](https://lynote.ai/)
2.  **输入** 粘贴文本或上传 PDF 文档。
3.  **点击** 生成，即可检查文本的ai率。

## 🎯 适用场景
* **学术研究:** 快速阅读 PDF 论文，使用 AI Detect 自查文稿。

* **内容创作:** 使用 AI Humanizer 优化文案，确保内容原创性。
