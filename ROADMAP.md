# 🗺️ Product Roadmap (产品路线图)

我们是一个认真做产品的团队。我们的愿景是让学习更简单、更高效。以下是我们的开发计划，欢迎社区投票反馈！

## 🚀 当前进度 (Current Status)
### [x] V0.1 - V0.5: 基础能力构建 (Live)
已上线和即将上线功能，主要集中在内容检测与视频解析基础能力。
- [x] **Core:** 网站基础版本上线 (多语言支持)
- [x] **Feature:** AI Detect (AI 内容检测，支持主流 LLM 模型)

即将上线：
- [x] **Feature:** AI Humanizer (文本人性化重写)
- [x] **Feature:** YouTube 文本转录 (带时间戳)
- [x] **Extension:** 谷歌插件 (YouTube Transcript 获取)

### [ ] V1.0: 学习闭环与多模态 (In Progress)
正在开发中，预计近期上线。重点在于"规划"与"整理"。
- [ ] **Feature:** AI 学习助手 (学习路径规划)
- [ ] **Feature:** 多文档/多源 AI 总结 (RAG 技术支持)
- [ ] **Feature:** 广泛的文件导入 (PDF, PPT, Docx, Epub, 实时录音)
- [ ] **Feature:** 笔记编辑器 (基础编辑与插入功能)
